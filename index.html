Creating new instances of a class

Below is an example of creating a new class object using Javascript and creating new instances of that class.

Lets create a user class to easily create new user objects:

//creates the class to create new users
class User {
constructor(name, age){
this._name = name;
this._age = age;
}

get name(){
return this._name;
}

get age(){
return this._age;
}
};

Lets also make an array to store our users in and a function to add our new users to it:

//creates an empty array to store users
const users = []

//creates a function to add new users to the user array
const addUser = (user) => {
users.push(user);
}

Now we can create some new users and add them to our users array using our addUser function:

//creates new users using the above class
const newUser = new User('Homer Simpson', 34);
const newUser1 = new User('Peter Griffin', 30);
const newUser2 = new User('Michael Scott', 44);
const newUser3 = new User('Hank Hill', 48);

//adds users to the users array using the addUser function
addUser(newUser);
addUser(newUser1);
addUser(newUser2);
addUser(newUser3);

We can see that our new users have been added to the users array by logging it to the console:

//logs the users that are in the Users array
console.log(users);

Lets have some fun with the data by creating a way to choose a random user:

//selects a random number based on the number of users in the users array
let randomUser = Math.floor(Math.random() * users.length);

Now we can use that random user to complete our sentence introducing that user:

//grabs a random user from the users array and logs a sentence with the user's name and age.
console.log(`Hi my name is ${users[randomUser].name} and I am ${users[randomUser].age} years old.`);

See for yourself!
